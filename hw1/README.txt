How to run?
Before running hw1.R, change variables workd and filename to appropriate value. For hw1-d.R, change pathsvm's value to the local svmLight's installation pathname.

Use ctrl+A to highLight all text in textediter area, then use ctrl + enter to run the program. Look for the accuracy report for this trial in the console window
